# LoopLink — Smart Business Assistant

LoopLink helps small business owners track daily sales and expenses, get instant insights, and grow their profit with AI-powered advice.

## Getting Started

```bash
npm install
npm run dev
```

## Build for Production

```bash
npm run build
```

The output will be in the `dist/` folder, ready to deploy on Vercel, Netlify, or any static host.

## Tech Stack

- React 18 + TypeScript
- Vite
- Tailwind CSS
- shadcn/ui (Radix UI primitives)
- React Router v6
